Intro to Manufacturing Computation

Individual Assignment

Overview

The purpose of this exercise is to get used to working with shapely polygons and to use it to compute some manufacturing geometry. It is your job to determine how to compute the cut files using Python, shapely, and the foldable_robotics package

Rubric

DescriptionPoints
Steps152
Discussion48
Total200

Resources

  • The Shapely User Manual
  • Foldable robotics package repository on GitHub
  • foldable robotics package documentation

Instructions

Note 2: Please update your foldable robotics package with the following commands before starting:

pip uninstall foldable_robotics
pip install --force-reinstall foldable_robotics

Note: Display the result of each step inline with your code. This can be done in the following way:

a = Polygon(...put your arguments in here...)
a  #the result will plot in your jupyter notebook

  1. Import shapely.geometry, foldable.layer, foldable.laminate, and foldable.manufacturing modules for using various shapely geometries, Layer(), and Laminate() classes, as well as for performing advanced computations.

    import shapely.geometry as sg
from foldable_robotics.layer import Layer
from foldable_robotics.laminate import Laminate

  2. Create a square polygon a using a shapely polygon class, with length and width equal to 1.

    Note: try using the Polygon class itself. You can create a polygon by supplying a list of points.

    result = sg.Polygon([[x1,y1],[x2,y2],...])

  3. Initialize a new Layer() b using polygon a.

  4. Translate b by a quantity (1.1,0), union with b and save as d.

  1. Create a new geometry e by dilating d by a value of 0.2 (try the methods .buffer(), .dilate(), as well as the operator <<).
  2. Create a new geometry f by eroding e by the same value used in the previous step (try the methods .buffer(), .erode(), as well as the operator >>).
  3. Compose a new Laminate() g consisting of d,f,and d, in that order from bottom(first) to top.
  4. Using a for loop to index through all the layers of g, create a new Layer() h consisting of a union of all the layers of g.
  5. Find a function within the Layer() class that will turn your layer h directly into a new three-layer laminate j.
  6. dilate j by 0.25 and save as k.
  7. dilate j by 0.50 and save as l.
  8. Find a function within the Laminate() class that will find the bounding box of k and save it as m
  9. remove k from m and save as n
  10. remove g from j and save as o
  11. dilate o by .26 and save as p
  12. remove j and p from k and save as q. Use the .plot() method to view the result in more detail.
  13. combine g, n, and q, and save as r. Individually plot each layer of the resulting laminate (There is a helper method for this inside the laminate class.)
  14. dilate j by .02, subtract j from the result, and save as s.
  15. remove s from r and save as t. Use the .plot() method to view the result in more detail.

Discussion

For the purposes of this discussion, assume that g is the desired laminate geometry you wish to create…the “laminate mechanism”.

  1. Identify what happens when you dilate and then shrink d by .1.
    1. How could this functionality be useful?
    2. What are the smallest dilate/erode values where this behavior is seen, and how is it related to the gap between polygons?
  2. Does f look any different from h? What could be the purpose (considering manufacturing) of using g to generate h rather than just using f directly?
  3. Please explain the purpose of creating j and k from the perspective of the laminate manufacturing process you have recently observed?
  4. What is the manufacturing purpose of n?
  5. What is the manufacturing purpose of q?
  6. What does s represent, and what does it produce in the the resulting t, from a manufacturing perspective?
